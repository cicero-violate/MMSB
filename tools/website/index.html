‚ö†Ô∏è Current Limitations (TO_DO_FIX.md)

Known Issues

  • ext filter + JSON: ?ext=.rs&format=json returns HTML instead of filtered JSON
  • Sorting: All sort parameters are ignored
  • Statistics: ?stats=true returns 400 Bad Request
  • Metadata: ?metadata=true returns 400 Bad Request
  • Pagination: limit/offset/page parameters ignored
  • Pretty JSON: ?pretty=true has no effect
  • Type filtering: ?type=rust not supported (400 error)
  • Recursive + JSON: ?recursive=true&format=json returns text instead of JSON

üéì Best Practices for LLM Agents

DO:

  • ‚úÖ Use format=json for structured data retrieval
  • ‚úÖ Use search=term to find specific files quickly
  • ‚úÖ Use preview=true for quick file summaries before full read
  • ‚úÖ Start with root /mmsb/?format=json to understand structure
  • ‚úÖ Combine working parameters: /mmsb/?search=config&format=json

DON'T:

  • ‚ùå Don't expect sort= parameters to work
  • ‚ùå Don't use stats=true or metadata=true (will error)
  • ‚ùå Don't rely on limit or offset for pagination
  • ‚ùå Don't expect ext=.rs&format=json to work together
  • ‚ùå Don't use recursive=true&format=json (returns text)

üí° Tips for Efficient Navigation

  1. Start broad, narrow down: Begin with /mmsb/?format=json, then drill into specific directories
  2. Use search liberally: The search function is fast and works well - use it!
