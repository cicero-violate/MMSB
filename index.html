Essential URLs

  • JSON listing: /mmsb/src?format=json - Machine-readable directory data

üéì Best Practices for LLM Agents

DO:

  • ‚úÖ Use format=json for structured data retrieval
  • ‚úÖ Use search=term to find specific files quickly
  • ‚úÖ Use preview=true for quick file summaries before full read
  • ‚úÖ Start with root /mmsb/?format=json to understand structure
  • ‚úÖ Combine working parameters: /mmsb/?search=config&format=json

DON'T:

  • ‚ö†Ô∏è Be aware that recursive=true&format=json is supported but may be slow for large directories
  • ‚ö†Ô∏è Use reasonable limit values to avoid overwhelming responses
  • ‚ö†Ô∏è Always include format=json when using API features like sort, stats, or metadata

üí° Tips for Efficient Navigation

  1. Start broad, narrow down: Begin with /mmsb/?format=json, then drill into specific directories
  2. Use search liberally: The search function is fast and works well - use it!
  3. Preview before reading: For documentation files, use ?preview=true to see if it's relevant
  4. Understand limitations: Don't waste time trying broken features - stick to working parameters
  5. Combine wisely: search + format=json works, but ext + format=json doesn't
