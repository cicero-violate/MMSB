## How to Extend
1. Copy the section template below into the appropriate file (or a new role file if needed).
2. Keep instructions concise; link to logs/scripts rather than duplicating.
3. When a playbook changes, add a short note in `TASK_LOG.md` so others know to re-read it.

### Template
```
# <Role> Agent Playbook
## Charter
## Tool Belt
## Standard Runbooks
## Reporting
## Open Questions / TODOs
```
